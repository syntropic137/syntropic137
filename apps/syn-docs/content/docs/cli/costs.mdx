View cost tracking data for sessions and executions

## `syn costs execution`

Show detailed cost breakdown for an execution

```bash
syn costs execution <execution-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `execution-id` | `text` | Yes | Execution ID |

---

## `syn costs executions`

List cost data for workflow executions

```bash
syn costs executions [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--limit`, `-n` | `text` | `50` | Max results |

---

## `syn costs session`

Show detailed cost breakdown for a session

```bash
syn costs session <session-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `session-id` | `text` | Yes | Session ID |

---

## `syn costs sessions`

List cost data for sessions

```bash
syn costs sessions [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--execution`, `-e` | `text` | --- | Filter by execution ID |
| `--limit`, `-n` | `text` | `50` | Max results |

---

## `syn costs summary`

Show aggregated cost summary

```bash
syn costs summary
```
