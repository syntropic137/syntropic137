List and inspect agent sessions

## `syn sessions list`

List agent sessions

```bash
syn sessions list [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--workflow`, `-w` | `text` | --- | Filter by workflow ID |
| `--status`, `-s` | `text` | --- | Filter by status |
| `--limit`, `-n` | `text` | `50` | Max results |

---

## `syn sessions show`

Show detailed session information

```bash
syn sessions show <session-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `session-id` | `text` | Yes | Session ID |
