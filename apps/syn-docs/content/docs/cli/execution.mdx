List and inspect workflow executions

## `syn execution list`

List all workflow executions

```bash
syn execution list
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--status`, `-s` | `text` | --- | Filter by status |
| `--page` | `text` | `1` | Page number |
| `--page-size` | `text` | `50` | Items per page (max 100) |

---

## `syn execution show`

Show detailed information about a single execution

```bash
syn execution show <execution-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `execution-id` | `text` | Yes | Execution ID |
