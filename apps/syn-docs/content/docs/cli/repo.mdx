Manage repositories and view observability data

## `syn repo activity`

Show recent execution activity for a repository

```bash
syn repo activity <repo-id> [options]
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `repo-id` | `text` | Yes | Repository ID |

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--limit`, `-n` | `text` | `20` | Max results |

---

## `syn repo list`

List repositories

```bash
syn repo list [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--org`, `-o` | `text` | --- | Filter by organization |
| `--system`, `-s` | `text` | --- | Filter by system |

---

## `syn repo register`

Register a repository

```bash
syn repo register [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--url`, `-u` | `text` | --- | Repository URL (owner/repo or full URL) |
| `--system`, `-s` | `text` | --- | Assign to system ID |
| `--org`, `-o` | `text` | --- | Organization ID |
