Manage systems and view system observability

## `syn system activity`

Show recent execution activity for a system

```bash
syn system activity <system-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `system-id` | `text` | Yes | System ID |

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--limit`, `-n` | `text` | `20` | Max results |

## `syn system create`

Create a new system

```bash
syn system create
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--name`, `-n` | `text` | --- | System name |
| `--description`, `-d` | `text` | --- | System description |
| `--org`, `-o` | `text` | --- | Organization ID |
