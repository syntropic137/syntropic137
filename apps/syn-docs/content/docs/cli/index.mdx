---
title: CLI Reference
description: Command-line interface for Syntropic137.
---

The `syn` CLI provides commands for managing workflows, agents, and the Syntropic137
platform from your terminal.

## Installation

The CLI is included when you install Syntropic137:

```bash
uv sync
syn --help
```

## Commands

### `syn workflow`

Manage and trigger workflows.

```bash
# List workflows
syn workflow list

# Run a workflow
syn workflow run --name <workflow-name>

# Get workflow status
syn workflow status <execution-id>
```

### `syn agent`

Manage agent sessions.

```bash
# List active sessions
syn agent list

# View session details
syn agent show <session-id>
```

### `syn config`

Manage configuration.

```bash
# Show current config
syn config show

# Validate configuration
syn config validate
```

## Global Options

| Option | Description |
|--------|-------------|
| `--help` | Show help message |
| `--version` | Show version |
| `--api-url` | Override API server URL |
| `--format` | Output format (`json`, `table`) |
