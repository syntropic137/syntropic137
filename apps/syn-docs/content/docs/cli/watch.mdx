Stream live execution events via SSE

## `syn watch activity`

Stream live global activity across all executions

```bash
syn watch activity
```

---

## `syn watch execution`

Stream live events for a specific execution

```bash
syn watch execution <execution-id>
```

**Arguments:**

| Name | Type | Required | Description |
|------|------|----------|-------------|
| `execution-id` | `text` | Yes | Execution ID to watch |
