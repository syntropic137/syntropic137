GitHub App integration

## `syn github repos`

List repositories accessible to the GitHub App

```bash
syn github repos [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--installation`, `-i` | `text` | --- | Filter by installation ID |
| `--include-private` | `boolean` | --- | Include private repos (default: true) |
