Global system insights and cost analysis

## `syn insights cost`

Show global cost breakdown

```bash
syn insights cost [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--days`, `-d` | `text` | `30` | Number of days to look back |

---

## `syn insights heatmap`

Show activity heatmap over time

```bash
syn insights heatmap [options]
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--days`, `-d` | `text` | `14` | Number of days to show |

---

## `syn insights overview`

Show global overview of systems and health

```bash
syn insights overview
```
