View aggregated workflow and session metrics

## `syn metrics show`

Show aggregated metrics (optionally filtered by workflow).

```bash
syn metrics show
```

**Options:**

| Flag | Type | Default | Description |
|------|------|---------|-------------|
| `--workflow`, `-w` | `text` | --- | Filter by workflow ID |
